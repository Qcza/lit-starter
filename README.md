# lit-starter

Starter template for the [lit](https://lit.dev/) based projects.

## Development
Run the app in development mode
```text
npm start
```

## Build
Build the app
```text
npm run build
```

## Tests
Run unit tests
```text
npm test
```

Run unit tests and watch
```text
npm run test:watch
```

## Linters
Run linters and prettier
```text
npm run lint
```

Run prettier
```text
npm run prettier
```

Run eslint
```text
npm run lint:eslint
```

Run lit-analyzer
```text
 npm run lint:lit-analyzer
```
